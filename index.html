Node 地下铁沙龙 #4

深入实践中的 Node.js

又是一年春暖花开的时节，Node地下铁迎来了第四场活动。

这次活动我们相聚在六朝古都——南京。

南京作为一个软件开发的名城，拥有庞大的企业级开发的软件产业。

而 Node.js 作为新兴的互联网开发技术，能否在企业级开发上占据一席之地呢？

我们这次邀请到几位有实战经验的专家，跟我们分享他们的想法和思路。

快来一起相约玄武湖畔吧！

查看视频
PPT

元 彦 @淘宝

深入 Node.js 组件化的 UI 渲染

本分享将介绍组件化的 UI 渲染以 React 为代表可以使 React 运行在 Node.js 进行渲染直出，解决了传统 SPA 应用的首屏性能低 与 SEO 弱两大问题，同时对比传统的模版化渲染方案，各自的利弊。

PPT

亦 才 @淘宝

Java 迁移 Node.js 实践分享

结合淘宝微海报的案例，分享 Java 迁移 Node.js 项目期间的一些经验与心得，总结一些迁移要点以及 Node.js 企业级应用开发规范。

PPT

黄一君 @途牛

如何定位线上 Node 进程的运行时错误

Node.js 单进程的模型下，健壮性虽然能由PM2等守护工具最后兜底一把，但是对于出现的阻塞很多时候都是无能为力的。分享基于 V8 引擎的底层监控实践，来定位到线上进程的异常点。

PPT

响马·刘琥 @第三极

Fib.js 和 Node.js

分享 Fib.js 是如何结合 V8 与 Fiber，解决回调地狱问题，并能够提供更强劲的并发能力。同时也会介绍如何在 Fib.js 生态上去兼容和使用 Node.js 的组件。
